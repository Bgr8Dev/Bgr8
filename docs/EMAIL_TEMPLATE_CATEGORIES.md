# Email Template Categories

All email templates in the Bgr8 system are categorized for easy identification and filtering in the Template Manager.

## Category Types

The system uses 6 main categories:

1. **`announcement`** - Celebratory, welcome, and important announcements
2. **`newsletter`** - Regular communications, newsletters, and educational content
3. **`notification`** - Transactional notifications and status updates
4. **`invitation`** - Invitations, requests, and calls to action
5. **`reminder`** - Reminders and follow-ups
6. **`custom`** - Custom templates created by admins

## Category Usage in Template Manager

The Template Manager allows you to:
1. **Filter by category** - Use the category dropdown to filter templates
2. **Search across categories** - Search works across all categories
3. **View category badges** - Each template displays its category badge
4. **Sync by category** - Built-in templates show their category when syncing

## Best Practices

- **Announcements**: Use for positive, celebratory messages
- **Newsletters**: Use for regular, scheduled communications
- **Notifications**: Use for transactional, status update emails
- **Invitations**: Use when requesting action or participation
- **Reminders**: Use for follow-ups and time-sensitive reminders
- **Custom**: Use for platform-specific templates not covered by built-in categories

## Filtering Tips

- Filter by "Announcements" to find all welcome and celebratory emails
- Filter by "Reminders" to find all reminder templates
- Filter by "Notifications" to find transactional emails
- Use search + category filter together for precise results
